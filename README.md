# add-hacktoberfest-label-to-repositories

> one-of script that adds hacktoberfest label to Octokit JS/TS repositories

## Usage

Add "hacktoberfest" label

```
GITHUB_TOKEN=<token here> node index.js
```

Remove "hacktoberfest" label

```
GITHUB_TOKEN=<token here> node index.js
```
